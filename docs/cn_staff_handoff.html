Upload & schedule the Chinese versions

Claude builds both Bilibili versions from one command — titles, subs, dub, thumbnail, description, all done. Two things are left, and they're both yours. This page tells you exactly what to check for each.

📍 Where to find your work: Notion → Chinese database → "Whose turn" board. Any card sitting under Reviewer: review draft is ready for you — nothing there needs a person before you touch it.

Your two jobs

1

Upload both drafts to Bilibili

  1. Open the row, open its Drive folder link → everything's in /CN/.
  2. Pull {id}_ensub.mp4 (英配中字) and {id}_cndub.mp4 (中配), plus the cover image.
  3. Titles are already written for you — copy V1 from # CN title on the page straight into the 英配中字 upload, and V2 straight into the 中配 upload. Don't retype or reformat them.
  4. Paste # CN description exactly as written. Don't shorten it, don't move anything — the sponsor link's position is deliberate (see "already decided" below).
  5. If the page has a # CN ad disclosure section, follow it — see the watch-for box.
  6. Submit both as drafts.
2

Schedule both

  1. Set the same publish time on Bilibili's built-in scheduler for both drafts.
  2. That's it. Claude reads the BV link and scheduled date back into Notion on its own afterward — you don't paste anything back.

Watch for: # CN ad disclosure. Only appears when a video has a sponsor. It names the exact boxes to tick on Bilibili's ad-declaration step — usually all three: 内容, 口播, 简介. If the row has one, tick what it says, no more and no less.

Already decided — don't second-guess these

  • ✓Title format. 英配中字 gets the Chinese title and the English title in brackets; 中配 is Chinese-only. Both already end in a 【 】-bracketed tag so they read correctly as two distinct uploads, not a re-upload of each other.
    V1 · 你的饮食,可能错了一辈子（You've been eating wrong your whole life）【英配中字】
    V2 · 你的饮食,可能错了一辈子 【中配】
  • ✓Where the sponsor link sits. Always in the middle of the description, after the main summary paragraph — never the first line. A description that opens with a promo link reads as spam, so this is fixed on purpose.
  • ✓Whether a video needs ad disclosure. Already decided by the Contains ads? checkbox — if it's ticked, the disclosure section is already written for you.

If something's off, flag it — don't fix it yourself

  • A file is missing from /CN/, or a video's length looks wrong next to the source.
  • Contains ads? is ticked but there's no # CN ad disclosure section (or vice versa).
  • The title or description reads oddly, has a mistranslation, or a term looks inconsistent with past videos.
  • Anything about a row you're just not sure about.

Message Wayne rather than editing around it — most of these trace back to something upstream that's worth fixing once, not per-video.
